# AutoReadme

Auto-generate meaningful code comments and README documentation as you code.

## Features

- **Auto-commenting**: Smart comments added automatically when you save
- **README generation**: Creates and updates README.md for your project
- **Multiple languages**: TypeScript, JavaScript, Python, Java, Go, Rust, C/C++, C#
- **One-time setup**: Just login and approve documentation for your project

## Getting Started

1. Install the extension
2. Click "AutoReadme: Login" in the status bar
3. Open a project folder
4. Accept the "Document this project?" prompt
5. Code normally - comments appear automatically on save!

## Settings

- `autoreadme.apiEndpoint`: Backend API URL
- `autoreadme.commentStyle`: Comment style (short, explanatory, pr-review)
